# Vocab
Vocabulary Notebook android app

A vocabulary notebook application.
Tasks: 
1. Use the free-tier Google Gemini api to get the Korean translation of the word and example sentences.
If there is more than one translation, get the example sentences for each translation.
These example sentences should be in English.
2. Use the free-tier Google Gemini api to get the opposite of the word.
If there is more than one opposite, get the opposite for each opposite.
These opposites does not need example sentences.
Only needs the opposite word and its Korean translation.
3. Store the words in a local database.
4. Display the words in a list.
List should be sorted by the word.
List will display both word and its Korean translation.
If the word has more than one translation, display all of them.
If the word in the list is clicked, display the Korean translation and example sentences.
5. Allow the user to search for a word.
6. Allow the user to delete a word.
7. Allow the user to edit a word.
8. Allow the user to add a word.
9. Save time when word is added.
10. Allow the user to sort words by alphabetically or added time.
11. Allow the user to sort words by alphabetically or added time in reverse order.
12. Use Google Drive API to backup the words.
13. Use Google Drive API to restore the words.
14. Use Google Drive API to sync the words.
15. If user is using Android, allow the user to use widget to add a word. 
16. The widget will show the random few words that has been added.
17. If widget is clicked, it will open the app.
18. If specific word is clicked, it will open the word details.

Tech Specs:
1. Use Compose multiplatform.
2. Use SQLDelight to store the words.
3. Target platform: Android, Windows
4. Gemini API
5. Google Drive API
